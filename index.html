About us
A college project designed as a realistic learning platform.
W2L Websites was created as a structured college project to demonstrate how a real educational platform could look and feel. The idea is to go beyond a simple static page and show a complete environment: a home page, tutorials, references, FAQ, and simulated authentication.

The design follows a dark, modern style with clear sections and consistent components. Each page has a specific purpose: the home page introduces the platform, the tutorials page simulates video lessons, and the references pages act like quick documentation.

All of this is built in a single HTML file for easier review, but the structure is similar to what you would find in a multi-page application. This makes it ideal for a college project because it shows layout, navigation, interactivity, and content organisation in one place.
Frequently Asked Questions
A page of questions that could realistically appear for students using this platform.
Do I need any experience before using W2L Websites? +
No. The platform is designed so that you can start from zero. The first tutorials explain what HTML and CSS are, and how a basic page is structured. You can then move gradually into JavaScript and database concepts.
Is this a real online platform or a college project? +
This is a college project that imitates a real online learning platform. The goal is to show how such a system could be organised: navigation, tutorials, references, FAQ, and account pages, all in one consistent design.
Can I actually log in and save progress? +
The sign up and log in pages are front-end mockups. They are included to demonstrate how forms and authentication screens would look, but they are not connected to a real backend in this version.
Where can I find quick syntax reminders? +
The References section is designed exactly for that. You can open HTML, CSS, JavaScript, MySQL and script references and quickly scan through the most important elements, properties and commands.
Can this project be extended in the future? +
Yes. The structure is ready to be expanded with more tutorials, more detailed reference pages, and a real backend if needed. It can grow from a college project into a more advanced portfolio piece.
